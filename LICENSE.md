## 🐛 Common Issues and Fixes

### ❌ GUI doesn't appear after execution
**Fix:** The panel may be rendering off-screen. Rejoin the server
and execute the script immediately after your character loads in.
Minimum recommended resolution is 1280×720 on PC.

### ❌ Auto rebirth not triggering at threshold
**Fix:** Update to v3.4.1 — this version contains a fix for the
rebirth UI structure change introduced in the March 2026 game
update that caused earlier versions to miss the trigger.

### ❌ Auto dupe pet produces no duplicates
**Fix:** Ensure you have at least one pet equipped before enabling
the **muscle legends script auto dupe pet** module. The duplication
system requires an active pet as its source reference.

### ❌ Auto treadmill stops after a few minutes
**Fix:** The server may have reset the treadmill object. The
**muscle legends script auto treadmill** module in v3.4.1 includes
an auto-reconnect loop that re-attaches to the treadmill after
any server-side reset.

### ❌ Auto kill not targeting enemies
**Fix:** Some game zones restrict combat interactions. Move to a
designated combat area and re-toggle the **auto kill muscle legends
script** module. Check that enemy detection range is set above 20
units in the settings panel.

### ❌ Script crashes on Android executor
**Fix:** Update your executor to the latest version. Arceus X
and Delta are the most stable options for the mobile build.
Restart both Roblox and the executor before retrying.

### ❌ "HttpGet failed" error on load
**Fix:** Verify your internet connection and confirm that HTTP
request permissions are enabled in your executor's settings panel.
